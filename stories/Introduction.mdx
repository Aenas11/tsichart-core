# TSIChart Component Library

Welcome to the TSIChart Storybook! This interactive documentation showcases the time series visualization components available in the TSIChart library.

## ğŸ¯ What is TSIChart?

TSIChart is a powerful, interactive time series charting library built with TypeScript and D3.js. It provides rich, customizable charts specifically designed for time series data visualization with features like:

- **Interactive Time Navigation**: Brush selection, zoom, pan
- **Multi-Series Support**: Display multiple data series with different styles
- **Advanced Tooltips**: Rich hover information with formatting options
- **Markers & Annotations**: Add custom markers to highlight important events
- **Responsive Design**: Charts adapt to container size automatically
- **Accessibility**: Built-in keyboard navigation and screen reader support
- **Theming**: Light and dark themes with customization options
